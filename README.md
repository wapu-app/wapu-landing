# Wapu Landing

Landing oficial de **Wapu CLI + Wapu API**.

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Desarrollo local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```

## Deploy

Deployado en Vercel.

- Producción: https://wapu.shiafu.com
- Fallback: https://wapu-landing.vercel.app
